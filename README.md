# Books

A repository for technical books and documentation built with [Quarto](https://quarto.org/).

## Development

```bash
quarto preview
```

The preview updates automatically when you edit files.

## Deployment

Push to `main` branch triggers GitHub Actions to build and deploy to `gh-pages`.

## Agentic Writing

Use `/book-writer` skill in [Claude Code](https://docs.anthropic.com/en/docs/claude-code) for agentic book writing.
